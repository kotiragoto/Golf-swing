# スイングスコープ ⛳

ゴルフスイング動画をブラウザだけで解析するアプリ。MediaPipe Pose（骨格検出）で体の動きに線を重ね、回転角・捻転差・軸ブレなどを定量評価します。**動画は端末内で処理され、どこにもアップロードされません。**

## 使い方

### 動画解析モード
1. `index.html` を開く（動画解析だけならダブルクリックで開いてもOK）
2. スイング動画（1スイング・10秒前後推奨）をドロップ。**正面・後方（飛球線後方/DTL）どちらの画角も対応**（自動判定、手動切替可）
3. 自動で全フレーム解析 → 骨格線つきでスロー再生・コマ送り
4. アドレス/トップ/インパクト/フィニッシュを自動検出（チップから手動修正可）

**表示される指標**
- 肩の回転 / 腰の回転 / X-Factor（捻転差）… 3Dワールド座標から算出（画角非依存）
- 前傾角、テンポ（バックスイング:ダウンスイング比、理想目安 3:1）
- 正面: 頭の横ブレ(cm)・腰のスウェー(cm) ／ 後方: 頭の上下動(cm)・アーリーエクステンション(腰の前後移動, cm)
- 総合スコアと項目別アドバイス

### リアルタイムモード
カメラを開くと骨格線が自動表示されます。構えて「アドレス基準セット」を押すと回転量がライブ表示。「録画して解析」で撮ったスイングをそのまま動画解析に回せます。

### 履歴
解析が完了すると自動で端末内（IndexedDB）に保存されます（最新50件）。履歴タブから過去の解析を一覧でき、**骨格リプレイ**・回転角チャート・評価レポートをいつでも見返せます。プライバシーと容量のため**映像そのものは保存されません**（骨格データ+計測値+サムネイルのみ、1件あたり数百KB）。

⚠ カメラは `file://` では動かないブラウザがあります。ローカルサーバー経由で開いてください:

```bash
cd ゴルフスイング
python3 -m http.server 8931
# → http://localhost:8931
```

スマホで使う場合は Vercel 等に置くと便利です（https必須のため）。単一HTMLなのでそのままデプロイできます。

## 注意
- 評価値は正面撮影を想定した目安です（プロ計測器の代替ではありません）
- iPhoneのHEVC(H.265)動画はChromeで再生できないことがあります → Safariで開くか「互換性優先」で撮影
- 検出モデル・WASMはCDNから読むため初回はネット接続が必要です
